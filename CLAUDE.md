# Moneyball App

## Project Overview
McKesson Brands product category prioritization dashboard

## Design Rules
- Always use light/white theme
- McKesson brand colors: blue #0066CC
- Never use dark backgrounds

## Data Structure
- 57 Level 2 categories
- Key metrics: MB GP$, MB GP%, MMS GP$, penetration, coverage
- Tiers 1-4 based on priority scoring

## Features Already Built
- Strategic Matrix scatter chart
- Portfolio Map
- GP Ranking
- Data Table

## Preferences
- Always use Recharts for charts
- Keep components modular
- Comments in plain English not technical jargon
